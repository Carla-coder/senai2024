Aula 006 - React BootStrap
React Bootstrap é uma biblioteca que oferece componentes React que implementam a biblioteca de interface do usuário Bootstrap. Com o React Bootstrap, os desenvolvedores podem usar componentes como Navbar, Button, Card, Form, Modal e muitos outros, sem precisar lidar com a integração manual do Bootstrap em seus aplicativos React

Biblioteca
https://react-bootstrap.netlify.app/

Iniciando projeto com React Bootstrap
npx create-react-app my-app
cd my-app
depois vamos instalar as dependencias

yarn add react-bootstrap bootstrap
yarn add react-router-dom
yarn add @babel/plugin-proposal-private-property-in-object
*************************************************************************
Uso do Bootstrap no React - Native

1. criar pasta
2. abrir VSC
3. Instalar as dependências conforme tutorial
4. depois de tudo instalado abrir com yarn run
